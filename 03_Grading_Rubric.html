Evaluates the student's ability to correctly install and configure the Windows operating system.

20 Points
Criteria Excellent
Full Score		 Proficient
Good		 Developing
Partial		 Beginning
Minimal		 Score
BIOS/UEFI Boot Setup
(max 4 pts)
 Correctly set USB as boot device; accessed BIOS without assistance. (4 pts) Accessed BIOS with minor guidance; boot order set correctly. (3 pts) Required significant teacher assistance to access BIOS. (2 pts) Unable to access BIOS or set boot priority. (0–1 pt)
Partitioning
(max 6 pts)
 Created precise partitions; left exactly the right amount of space for Linux. (6 pts) Partitioned correctly with minor sizing error (within 5GB). (max 6 pts)

(max 4 pts)
 User account created properly; system settings configured. (4 pts) User account created; minor settings skipped. (3 pts) Basic setup done with some missing configuration. (2 pts) User configuration not completed. (0–1 pt)

Category 2: Linux OS Installation

Evaluates the student's ability to correctly install a Linux distribution alongside Windows.

20 Points
Criteria Excellent (Full) Proficient (Good) Developing (Partial) Beginning (Minimal) Score
Bootable USB & Startup
(max 4 pts)
 Booted from Linux USB correctly; installer launched without issues. (4 pts) Minor difficulty booting; resolved independently. (3 pts) Needed assistance to boot from USB. (2 pts) Could not boot from Linux USB. (0–1 pt)
Partition Setup
(max 6 pts)
 Proper root, swap, and /home partitions created in unallocated space. (6 pts) Partitions set up correctly but missing /home or undersized. (4–5 pts) Partitions set up with errors; required correction. (2–3 pts) Did not partition correctly; overwrote Windows or failed. (0–1 pt)
Successful Installation
(max 6 pts)
 Linux installed flawlessly; GRUB installed; system boots. (6 pts) Linux installed with 1 issue resolved with minimal help. (4–5 pts) Linux installed but significant errors required teacher help. (2–3 pts) Linux did not install or GRUB was missing. (0–1 pt)
User & System Setup
(max 4 pts)
 Created user account, set timezone, and configured terminal access. (4 pts) User account created; some settings not configured. (3 pts) Minimal setup; only user account created. (2 pts) Setup incomplete or not done. (0–1 pt)
SUBTOTAL — Linux OS Installation

Category 3: Dual-Boot Setup & Verification

Evaluates successful configuration and demonstration of dual-boot functionality via GRUB.

15 Points
Criteria Excellent (Full) Proficient (Good) Developing (Partial) Beginning (Minimal) Score
GRUB Boot Menu
(max 5 pts)
 GRUB appears correctly; shows both Windows and Linux options. (5 pts) GRUB appears but with minor display issue; both OSes selectable. (3–4 pts) GRUB configured but only one OS selectable. (2 pts) GRUB not installed or not visible. (0–1 pt)
Boot into Windows
(max 5 pts)
 Successfully boots Windows from GRUB without errors. (5 pts) Boots Windows with minor slowness / startup issue. (3–4 pts) Windows boots but shows errors or requires repair. (2 pts) Cannot boot Windows from GRUB. (0–1 pt)
Boot into Linux
(max 5 pts)
 Successfully boots Linux from GRUB without errors. (5 pts) Boots Linux with minor issue resolved quickly. (3–4 pts) Linux boots but with display or filesystem errors. (2 pts) Cannot boot Linux from GRUB. (0–1 pt)
SUBTOTAL — Dual-Boot Setup

Category 4: Subnetting & IP Configuration

Evaluates the ability to design a correct subnet and configure static IP addresses on all computers.

20 Points
Criteria Excellent (Full) Proficient (Good) Developing (Partial) Beginning (Minimal) Score
Subnet Calculation
(max 8 pts)
 Correct network address, subnet mask, broadcast, host range, and CIDR notation. (8 pts) 1–2 fields have minor errors; overall logic is correct. (6–7 pts) Subnet calculation has multiple errors; partially correct. (3–5 pts) Subnet calculation is mostly incorrect or missing. (0–2 pts)
Static IP Assignment
(max 6 pts)
 All computers have correct, non-conflicting IPs within the valid host range. (6 pts) IPs assigned correctly on most computers; 1 minor conflict. (4–5 pts) IPs partially configured; some machines not in the correct range. (2–3 pts) IPs not correctly assigned on most computers. (0–1 pt)
Physical Connection
(max 6 pts)
 LAN cables or switch connected properly; link lights are active on all ports. (6 pts) Physical connection made but one port needed re-seating. (4–5 pts) Connection attempted but unstable or incomplete. (2–3 pts) Physical connection not established. (0–1 pt)
SUBTOTAL — Subnetting & IP Configuration
📡

Category 5: Ping Test & Message Sending

Evaluates network connectivity testing and successful peer-to-peer communication between machines.

15 Points
Criteria Excellent (Full) Proficient (Good) Developing (Partial) Beginning (Minimal) Score
Ping Test (Windows)
(max 5 pts)
 All 4 packets sent and received (0% loss) on both directions. (5 pts) Ping successful, minor packet loss (1 packet). (3–4 pts) Ping partially successful; high packet loss but some replies. (2 pts) Ping failed entirely (request timed out). (0–1 pt)
Ping Test (Linux)
(max 5 pts)
 Successful ping from Linux to Windows (or Linux-to-Linux) using terminal. (5 pts) Ping successful with minor latency or 1 packet loss. (3–4 pts) Partial success; intermittent ping replies. (2 pts) ping command not attempted or all packets lost. (0–1 pt)
Message Sending
(max 5 pts)
 Message sent and received successfully on the other computer (verified live). (5 pts) Message sent but required minor troubleshooting (port/firewall). (3–4 pts) Message attempted but only partially delivered. (2 pts) Messaging not attempted or completely failed. (0–1 pt)
SUBTOTAL — Ping Test & Messaging
📝

Screenshots / Evidence
(max 5 pts)
 All 6 required screenshots submitted, clearly labeled and readable. (5 pts) 4–5 screenshots submitted; most are labeled. (3–4 pts) 2–3 screenshots submitted; labels missing or unclear. (2 pts) 0–1 screenshot submitted. (0–1 pt)
